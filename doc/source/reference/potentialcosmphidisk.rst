.. _cosmphidisk:

Cos(m phi) disk potential
=============================

Generalization of the :ref:`lopsided <lopsideddisk>` and
:ref:`elliptical <ellipticaldisk>` disk potentials to any m and to
allow for a break radius within which the radial dependence of the
potential changes from R^p to R^-p.
